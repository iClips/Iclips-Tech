Development Tools and Setup

- Install and run VS Code
- Download and Install NodeJS
- Open VS Code and open Extensions
- Search and install HTML CSS Support, Live Server, Code Runner and JavaScript (ES6) code snippets

Develop, Run and Test

- Go to File -> Add Folder to Workspace and select/create a folder and click Add
- Create and code your website pages
- Go to Run -> Run Without Debugging

Deploy Digital Solution

- Upload and deploy project to webserver
- Test on live enviroment
- Maintain project
